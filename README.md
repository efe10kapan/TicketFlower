# TicketFlower
TicketFlower | Sinema Yönetim ve Biletleme Platformu
Proje Özeti
TicketFlower, modern web standartları çerçevesinde sinema salonu operasyonlarını ve kullanıcı biletleme süreçlerini dijitalleştirmek amacıyla geliştirilmiş uçtan uca (full-stack) bir web uygulamasıdır. Proje, dinamik veri yönetimi ve kullanıcı dostu arayüz tasarımı ilkeleri üzerine inşa edilmiştir.

Teknoloji Yığını (Tech Stack)

Frontend Geliştirme

React.js & Vite: Performans odaklı, hızlı yüklenen tek sayfa uygulama (SPA) mimarisi.

Tailwind CSS: Responsive ve ölçeklenebilir arayüz tasarımı için yardımcı program öncelikli (utility-first) CSS çerçevesi.

Bileşen Odaklı Mimari: Yeniden kullanılabilir UI bileşenleri ile kod sürdürülebilirliği sağlanmıştır.

Backend Geliştirme

Node.js & Express.js: Asenkron veri işleme yeteneğine sahip sunucu tarafı çalışma ortamı.

REST API Tasarımı: İstemci ve sunucu arasında JSON formatında kesintisiz veri alışverişi.

Veri Yönetimi

MongoDB: Esnek veri şeması sunan, doküman tabanlı NoSQL veritabanı.

Mongoose: Nesne Veri Modelleme (ODM) kütüphanesi ile şema tabanlı veritabanı güvenliği.

Temel Fonksiyonel Özellikler

Kullanıcı Modülü: Film kataloglama, seans detaylandırma ve interaktif koltuk seçim sistemi.

Yönetim Paneli (Admin): CRUD işlemleri aracılığıyla film, salon kapasitesi ve bilet verilerinin dinamik yönetimi.

Entegrasyon: İstemci ve sunucu tarafının gerçek zamanlı veri senkronizasyonu.

Uyumluluk: Masaüstü ve mobil platformlar için optimize edilmiş responsive tasarım.

Proje Organizasyonu
Geliştirme standartlarına uygun olarak modüler bir yapıda kurgulanmıştır:

/frontend: Kullanıcı arayüzü bileşenleri ve state yönetimi.

/backend: API rotaları, veri modelleri ve veritabanı bağlantı konfigürasyonları.
